Program Arduino ini digunakan untuk menampilkan teks berjalan (scrolling text) pada LCD 16x2 menggunakan library Wire dan Adafruit_LiquidCrystal. Pada bagian awal, dibuat objek lcd dengan alamat default, lalu didefinisikan variabel teks yang berisi kalimat panjang dengan tambahan spasi di depan agar efek scrolling terlihat halus. Di fungsi setup(), LCD diinisialisasi dengan ukuran 16 kolom dan 2 baris, kemudian kata "QUOTE" ditampilkan di baris pertama sebagai judul. Selanjutnya pada fungsi loop(), program menggunakan perulangan for untuk menggeser potongan teks sepanjang 16 karakter dari variabel teks menggunakan substring, lalu ditampilkan di baris kedua LCD secara terus-menerus. Setiap pergeseran diberi jeda delay(100) agar animasi berjalan tidak terlalu cepat, sehingga menghasilkan efek teks berjalan dari kanan ke kiri.
