

Project ini menggunakan Arduino Uno dan LCD 16x2 dengan modul I2C untuk menampilkan teks berjalan secara horizontal pada baris kedua, sementara baris pertama menampilkan judul statis.

Deskripsi Program
Program ini menampilkan sebuah kutipan bijak: *"Di balik takdir yang pahit, ada hikmah yang manis"*. 
- Baris 1: Menampilkan teks statis "QUOTE" di posisi tengah.
- Baris 2: Menampilkan teks berjalan dari kanan ke kiri menggunakan fungsi `substring()` untuk menciptakan efek animasi yang halus.

Komponen yang Digunakan
1. Arduino Uno (atau jenis Arduino lainnya)
2. LCD 16x2
3. Modul I2C (PCF8574) - Untuk menghemat pin Arduino.
4. Kabel Jumper

 Rangkaian (Wiring)
| LCD I2C Pin | Arduino Pin |
|-------------|-------------|
| GND         | GND         |
| VCC         | 5V          |
| SDA         | A4 (SDA)    |
| SCL         | A5 (SCL)    |

Kode Program
Program ini membutuhkan library `LiquidCrystal_I2C`. Pastikan Anda sudah menginstalnya melalui Library Manager di Arduino IDE.

```cpp
#include <Wire.h> 
#include <LiquidCrystal_I2C.h>

// Inisialisasi alamat I2C 0x27 untuk LCD 16 baris 2 kolom
LiquidCrystal_I2C lcd(0x27, 16, 2); 

void setup() {
  lcd.init(); 
  lcd.backlight();
}

void loop() {
  String quote = " Di balik takdir yang pahit, ada hikmah yang manis";
  int len = quote.length();
  
  for (int pos = 15; pos >= -len; pos--) {
    
    lcd.setCursor(5, 0);
    lcd.print("QUOTE");
    
    if (pos >= 0) {
      lcd.setCursor(pos, 1);
      lcd.print(quote.substring(0, 16 - pos));
    } else {
      lcd.setCursor(0, 1);
      lcd.print(quote.substring(-pos, -pos + 16));
    }
    
    delay(300); // Kecepatan pergerakan teks (ms)
    lcd.setCursor(0, 1);
    lcd.print("                ");
  }
}
