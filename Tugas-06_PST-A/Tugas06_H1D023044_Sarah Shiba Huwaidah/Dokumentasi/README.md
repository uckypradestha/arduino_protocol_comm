# Dokumentasi Proyek I2C LCD

Folder ini memuat seluruh aset visual yang membuktikan bahwa sistem telah dirangkai dan berjalan dengan baik pada simulator Tinkercad.

## Daftar Aset
1. Skematik Rangkaian
   Menunjukkan penyambungan kabel antara Arduino Uno dan pin I2C pada LCD
   - GND terhubung ke GND
   - VCC terhubung ke 5V
   - SDA terhubung ke pin khusus SDA
   - SCL terhubung ke pin khusus SCL

2. Demo Simulasi (berupa gif)
   Rekaman layar yang menunjukkan hasil akhir saat program dijalankan, di mana tulisan QUOTE diam di atas dan teks berjalan dari kanan ke kiri di baris bawah.
