## Tujuan Direktori
Direktori ini berisi hasil pengerjaan tugas simulasi komunikasi Inter-Integrated Circuit (I2C) menggunakan Arduino Uno dan modul LCD 16x2. Sistem dirancang untuk menampilkan dua baris teks:
- **Baris 1 (Statis):** Menampilkan kata "QUOTE" tepat di tengah layar.
- **Baris 2 (Dinamis):** Menampilkan teks Quote yang berjalan (*scrolling*) dari ujung kanan ke ujung kiri secara berulang.

## Struktur Direktori
- `dokumentasi/` : Berisi bukti visual berupa gambar rangkaian circuit, rangkaian skematik dan animasi GIF hasil simulasi di Tinkercad.
- `source_code/` : Berisi file kode program Arduino (`.ino`) beserta penjelasan teknis cara kerja kodenya.
