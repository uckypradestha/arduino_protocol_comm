<h1>Template Laporan Tugas</h1>

<h2>Cara Menggunakan</h2>

Untuk menggunakan template ini dalam laporan Anda, fork repositori ini, ubah namanya sesuai dengan Modul Praktikum yang telah Anda praktikan, dan sesuaikan lisensinya sesuai kebutuhan.

Untuk menggunakan repositori ini dalam laporan Anda sendiri, cukup klik "Use this template" di bagian atas halaman ini.

Sebagai alternatif, Anda dapat melakukan clone repositori dan menyusunnya ulang sesuai kebutuhan. Melalui command line,

```
> git clone git@github.com:uckypradestha/arduino_protocol_comm.git
```

<h2>Layout</h2>

Repositori ini dibagi menjadi tiga direktori utama, yang sebagian besar memiliki subdirektori. Struktur ini dirancang agar mudah dinavigasi baik oleh manusia maupun komputer, sehingga memungkinkan pencarian file dan instruksi tertentu dengan cepat.

Di dalam setiap direktori terdapat file ```README.md``` yang merangkum tujuan dari direktori tersebut serta beberapa contoh jika diperlukan. Struktur ini mungkin tidak sepenuhnya sesuai dengan kebutuhan penggunaan Anda dan bisa saja perlu dimodifikasi.

Khusus untuk direktori Sourc Code, file ```README.md``` digunakan untuk menjelaskan Code yang telah Anda buat.

## Laporan Tugas

Ganti isi file README.md ini dengan penjelasan tentang kode yang telah dibuat mulai dari penggunaan Librari, Inisialisasi dan Looping.
