## Alur Pengumpulan Tugas

1. Fork repository ini
2. Clone ke lokal
3. Kerjakan tugas pada folder Tugas (Disesuaikan dengan Mata Kuliah dan Kelas)
4. Pada folder tugas buat folder dengan format `Tugas06_NIM_Nama` sebagai direktori untuk penyimpanan file tugas berupa:
   - `Scetch.ino` atau `main.cpp`
   - Folder Dokumentasi yang berisi rekaman dalam bentuk video ketika program dijalankan di hardware
   - File `README.md` yang digunakan untuk menjelaskan kode program
6. Commit dan push:
   - git add .
   - git commit -m "Modul X - Nama NIM"
   - git push
7. Buat Pull Request ke repository utama
